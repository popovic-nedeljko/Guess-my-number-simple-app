# Guess my number project

[Gues-my-number](https://guess-my-number-simple-app.netlify.app/)

Game application in which the goal is to guess the random generated number, from 1 to 25.

## NOTE SETTINGS

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Terminal:

1. Run npm install inside project root

```
'npm instal'
```

2. Start the server

```
'npm run start'
```

3. Play

```
Enjoy :)
```
